# Mahasiswa REST API

A simple RESTful API for managing student (mahasiswa) data built with Node.js, Express, PostgreSQL, and Zod for request validation. This project is created as a backend fundamentals learning project.

## Features

- CRUD Mahasiswa
- Request validation with Zod
- Bulk insert

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Zod
